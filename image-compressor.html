Cómo comprimir imágenes con Alfora

Selecciona una imagen, define el nivel de calidad y descarga una versión optimizada. Todo ocurre dentro de tu navegador.

1. Carga

Selecciona una imagen JPG, PNG, WebP o SVG.

2. Ajusta

Elige un nivel de calidad o utiliza uno de los ajustes rápidos.

3. Descarga

Compara el peso y guarda la versión comprimida.
